⚙️  CONFIGURATION

   Default Settings:
   • Symbol: BTC/USD (ID: 10028)
   • Quantity: 0.10
   • Timeframe: M15 bars
   • Strategy: MA Crossover (or DDQN if model provided)

   Customize:
   export CTRADER_BTC_SYMBOL_ID=10028
   export CTRADER_QTY=0.10
   export DDQN_MODEL_PATH=/path/to/model.pth
   export DDQN_DUAL_AGENT=1              # Use dual-agent (TriggerAgent + HarvesterAgent)
   export DDQN_ENABLE_TRAINING=1         # Enable online learning (default: 1)
   export DDQN_TRAIN_EVERY_N_BARS=5      # Train every N bars (default: 5)
   export DDQN_TRAIN_AFTER_TRADES=1      # Train after each trade (default: 1)
